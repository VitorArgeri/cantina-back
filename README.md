## Requisitos - Biblioteca Web
---

### Requisitos Funcionais
- Possibilita Login
- Possibilita diferenciação entre administrador e usuário comum
- Possibilita mostrar todos os alimentos
- Possibilita mostrar um determinado alimento pelo seu ID
- Possibilita atualização de alimentos
- Possibilita cadastro de alimentos
- Possibilita exclusão de alimentos
- Mostra uma mensagem de alerta caso o estoque esteja com poucos alimentos
- Possiblita o registro de vendas
- Possibilita o controle de estoque

---

### Requisitos Não-Funcionais
- Possibilita uma experiência de design simples ao usuário
- Possibilita segurança
- Possibilita clareza quanto as funcionalidades
